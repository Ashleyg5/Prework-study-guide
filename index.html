HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • box-shadow: adds shadow effects around an elements frame
  • width/margin/border all assign visual attributes to HTML through CSS
  • link rel="file name" href="file location": links CSS sheet to HTML file
  • .card: period preceding card designates this as a class selector. Any HTML element assigned to the class will gain these property assignments from CSS

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds the changed files in the working directory to the staging area of Github
  • git commit -m "message here": commits changes to the repository
  • git pull origin "branch name": ensures current branch is up-to-date before opening a pull request

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Within a variable there are values that have different data types ie. string, number, boolean, array, and object
  • Control flow is the order in which acomputer executes code in a script
  • Comments can be left on code using /* and */
  • An operator is a mathematical symbol that produces a result based on two variables ie. addition, subtraction, division, multiplication, assignment, strict equality, and does-not-equal
  • Conditionals are structures used to test if an expression returns true or not ie. if....else
  • Functions are sets of instructions that tells the computer how to perform a task
